# Зображення для Pronunciation Quest

Ця директорія містить зображення та графічні елементи для гри Pronunciation Quest.

## Типи зображень

У грі використовуються наступні типи зображень:
- Іконки досягнень
- Фонові елементи з британською тематикою
- Ілюстрації для окремих слів (опціонально)

## Рекомендації щодо зображень

- Формат: PNG або SVG для кращої якості
- Розмір: оптимізовані для веб (максимум 200-300 KB)
- Стиль: витримані в єдиному стилі дизайну
- Ліцензія: вільні для комерційного використання або з відповідним зазначенням авторства
